---
name: Document every custom skill
description: Every custom skill created for the project must have a corresponding doc file in docs/skills/
type: feedback
---

Ogni skill custom creata in `.claude/skills/` deve avere un file di documentazione corrispondente in `docs/skills/<nome-skill>.md`.

**Why:** L'utente vuole che le skill del progetto siano documentate per il team, non solo per Claude.

**How to apply:** Quando crei una nuova skill in `.claude/skills/`, crea sempre anche `docs/skills/<nome-skill>.md` con: cosa fa, quando si attiva, cosa controlla, dove si trova, perché esiste.
